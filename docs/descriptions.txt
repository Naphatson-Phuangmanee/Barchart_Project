class Graphic

เป็นคลาสที่อำนวยความสะดวก ในการวาดรูปทรงพื้นฐาน ลงบนหน้าจอ  ซึ่งประกอบด้วย วาดสี่เหลี่ยม วาดเส้นตรง และวาดตัวอักษร

Constructor ของ class Graphic รับอินสแตนซ์ของคลาส Turtle เพื่อใช้เป็นเครื่องมือในการวาดรูปทรงต่างๆ


class Bar

เป็นคลาสสำหรับวาดแท่งข้อมูล 1 แท่ง ของ barchart  คลาสนี้เรียกใช้ method ใน Class Graphic เพื่อการวาดรูปสี่เหลี่ยม และวาดตัวอักษร


class BarChart

เป็นคลาสที่ควบคุมการวาด ส่วนประกอบทุกส่วน ของ barchart  คลาสนี้รับข้อมูลรายละเอียดทั้งหมด ของ barchart และประมวลผล เพื่อหาขนาด และตำแหน่ง ที่เหมาะสม ของทุกส่วนประกอบ ซึ่งประกอบไปด้วย title, axes, paddings, gaps, และแท่งข้อมูลทุกแท่ง




โปรแกรมอนุญาตให้ผู้ใช้วาด barchart จากชุดข้อมูล (data set) ที่ประกอบไปด้วยหลายตัวอย่างข้อมูล (data point)  ใน 1 ตัวอย่างข้อมูล จะประกอบไปด้วยป้ายกำกับ (label) และตัวเลขที่บ่งบอกถึงค่าข้อมูลนั้น (value)  โดย 1 ตัวอย่างข้อมูลนี้ จะถูกนำไปวาดเป็นแท่งข้อมูล 1 แท่ง

โปรแกรมอนุญาตให้ผู้ใช้ นำชุดข้อมูลเข้าสู่โปรแกรม ได้ 2 วิธี คือ

    1. ให้ผู้ใช้ระบุตำแหน่งของไฟล์ text ประเภท CSV ซึ่งเก็บ 1 ตัวอย่างข้อมูล (data point) ใน 1 แถว (line) ของไฟล์

    2. ให้ผู้ใช้ป้อนชุดข้อมูลด้วยตนเอง  โปรแกรมจะวนซ้ำเพื่อรับข้อมูลจากผู้ใช้ โดยโปรแกรมจะรับ 1 ตัวอย่างข้อมูล (data point) ใน 1 รอบของการวนซ้ำ โดยร้องขอให้ผู้ใช้กรอกป้ายกำกับ (label) และค่าของข้อมูล (value) นั้นๆ  โปรแกรมจะร้องขอให้ผู้ใช้ ระบุตำแหน่งของไฟล์ เพื่อบันทึกข้อมูลที่ป้อนเข้ามา

ข้อมูลที่ได้ จะถูกส่งให้ class BarChart ประมวลผล  class BarChart จะเรียกใช้ class Graphic เพื่อให้ช่วยในการวาดส่วนประกอบต่างๆ เช่นวาดเส้นตรง และวาดตัวอักษร  ส่วนการวาดแท่งข้อมูลแต่ละแท่งนั้น class BarChart  จะส่งต่อความรับผิดชอบ ให้ class Bar เป็นผู้วาด
